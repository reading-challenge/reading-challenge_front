# reading-challenge_front

독서 챌린지 웹 서비스 프론트입니다.

# 프로젝트 생성

Node.js 20.11.0 설치 후
npm init vue@latest

# 프로젝트 환경

Vue: 3.4.15, TypeScript
vite: 5.0.11
Node.js: 20.11.0

# 프로젝트 실행

npm install (프로젝트 처음 받았을 때 한 번만 해주면 됨)
npm run dev
